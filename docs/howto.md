
## Keybinds



### Scrolling down menus for autocomplete
| Mode    | Keybind        | Effect                          |
| :-----: | :------------: | :-----------------------------: |
| `n`     | `<C-n>`        | Move to [n]ext item in list     |
| `n`     | `<down-arrow>` | Move to next item in list       | 
| `n`     | `<C-p>`        | Move to [p]revious line in list | 
| `n`     | `<up-arrow>`   | Move to previous line in list   |




| Mode | Keybind        | Effect                          |
| :--: | :------------: | :-----------------------------: |
| `n`  | `za`           | Toggle fold at curreht line     |
  
### Searching

-   Type in the `/` symbol to initiate a search 
-   Type in the search string or regex
-   Hit `<Enter>` 
-   Press `n` to move to next occurent and `N` to move back
